Installation

Connect your application with Ultimate.

You can install Palzin APM package using composer:

1composer require ingress-it-solutions/ultimate-php

Server requirements

¶
  • PHP >= 7.2

Create an Palzin APM instance

¶

Be sure to include the composer autoload in your application. The first step is to create an instance of the Configuration class using the Ingestion Key.

1
2<?php
3require __DIR__ . '/../vendor/autoload.php';
4
5use Ultimate\Ultimate;
6use Ultimate\Configuration;
7
8// Create a configuration instance.
9$configuration = new Configuration('YOUR_INGESTION_KEY');
10
11// Pass the configuration to the Palzin APM constructor.
12$ultimate = new Ultimate($configuration);

Get a new Ingestion Key by signing up for Palzin APM (https://www.palzin.app/register) and creating a new application.
