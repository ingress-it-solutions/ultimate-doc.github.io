Upgrade Guide

Upgrading to 3.0 from 2.x

Estimated Upgrade Time: Five Minutes

¶

We attempt to document every possible breaking change. Since some of these breaking changes are in hidden parts of the library only a portion of these changes may actually affect your application.

Updating dependencies

¶

Update your ultimate-apm/ultimate-laravel dependency to ^3.0 in your composer.json file.

Update the packages in your application by:

1composer update

Change Palzin APM URL

¶

The url of Inspetor ingestion system was changed a few month ago, so if you have published the ultimate.php configuration file in the config directory be sure that the url parameter is set to: https://ingest.ultimate.dev

Change the env viariable name

In the previous package version you have installed the KEY to enable data transfer from your server to Palzin APM API in the application's environment file:

1ULTIMATE_API_KEY=xxx

The name of this environment variable was changed as ULTIMATE_INGESTION_KEY, so be sure to change the .env file as below:

1ULTIMATE_INGESTION_KEY=xxx

What's new

¶
  • A redesigned version of our Middleware to provide better support for http requests ignoring.
  • Add new segments using a callback in one line of code.
  • Some improvements on metrics collection.
  • Ignore Jobs
