Introduction

Welcome to the Palzin APM developer documentation

Palzin APM is a Code Execution Monitoring tool that helps software developers to identify bugs and bottlenecks in their application before users stumble onto the problem.

Installing our library in your application you will be able to visualize what your application is doing on every execution cycle and identify unexpected behavior at a glance.

Screen shot: Palzin APM

This documentation will help you to start working with Palzin APM in just a few minutes, as well as support if you get stuck.

Below are the supported technologies. Choose the one your application is built with.

Security

¶

All connections with Palzin APM API are secured by an industry-standard 256-bit HTTPS Encryption. Furthermore our data storage and analysis infrastructure is not accessible from the public internet.

Performance

¶

Palzin APM is designed to be completely transparent from your client point of view. It collects and sends data to our inspection API asynchronuosly. This means that it doesn't introduce any latency in your code execution.
